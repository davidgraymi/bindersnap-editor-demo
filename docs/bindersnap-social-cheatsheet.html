Voice & Copy — Do's & Don'ts
✓ Write this

"Your approval process is a mess. We fixed it."

✕ Not this

"Bindersnap offers advanced document governance workflows."

✓ Write this

"Kill the email approval chain. Finally."

✕ Not this

"Leverage our rule-based approval infrastructure to streamline compliance."

✓ Write this

"Every approval. Logged. Tamper-proof."

✕ Not this

"Bindersnap may help facilitate more efficient document sign-off processes."

✓ Write this

"contract_FINAL_v2_JanEdits_APPROVED(1).docx — Sound familiar? 😩"

✕ Not this

"Our AI-powered platform synergizes your document management ecosystem."

Typography Quick Reference
Headlines
Lora · Bold / SemiBold
All H1–H3. The "soul" of the brand. Always warm and authoritative. Italic for accent phrases.
Body copy and UI
Geist · Light / Regular / Medium
All body text, buttons, nav, labels. Clean and modern. Pairs with Lora without competing.
SECTION TAGS · CODE
Geist Mono · Regular / Medium
Section eyebrows, code snippets, file names, timestamps, badges. Technical and precise.
